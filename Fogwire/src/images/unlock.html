Unlocking Instructions

Step 1:   Locate the unlocked.txt file that contains the encrypted message.
  1. If you were sent a locked message via email, download the unlocked.txt attachment and save locked.txt to the C:\FogWire2\Locked_Docs folder


Step 2:   Set the appropriate fogcode
  1. If you received a locked message from another registered fogwire user, have that user communicate the fogcode.
  2. From the Lock pane, click New Fog code to set the appropriate Fog Code
  3. Return to the Unlock pane


Step 3:   From the Unlock panel, browse for and select the unlocked.txt file.
  1. Click Open to display the encrypted text in the Locked Text area.
  2. Click the Unlock button to decypher the text and display the unlocked message to the text area
  3. The unlocked message is saved to: C:\FogWire2\Locked_Docs\locked.txt


IMPORTANT NOTES:

  1. The folders: C:\FogWire2\Unlocked_Docs and C:\FogWire2\Locked_Docs and the files: C:\FogWire2\Unlocked_Docs\unlocked.txt and C:\FogWire2\Locked_Docs\locked.txt are required by FogWire.
  2. locked.txt and unlocked.txt are overwritten each time you unlock data.
  3. Locked documents can only be unlocked with the correct fogcode.
  4. Allow for appropriate time for unlocking files over 1MB.
  5. locked.txt and unlocked.txt are overwritten during each unlock operation.
